# PI Email Templates

Templates for requesting PI review of concept-set code lists. Use the one matching the
concept type. Fill in `[Last Name]` and `[Study Name]` from the project context.

## Diagnosis codes (ICD-9 / ICD-10)

Subject: [Study Name] — Diagnosis Code Review Needed

Dear Dr. [Last Name],

We're moving forward with the data extraction for [Study Name] and need your input before
we can proceed.

We've generated a list of diagnosis codes (ICD-9 and ICD-10) that may be relevant to your
study. Please review the attached spreadsheet and confirm which codes to include.

**What we need from you:**

1. Open the attached spreadsheet.
2. Review the diagnosis codes listed.
3. In the `Desired` column, change any code from `TRUE` to `FALSE` if it should be excluded.
   All codes are currently marked `TRUE` — you only need to update codes that shouldn't be included.
4. (Optional) Update the `Category` column if a code is incorrectly categorized.
5. Reply with the updated spreadsheet attached.

If all listed codes are appropriate, no changes are necessary. Feel free to add comments in
the spreadsheet for any codes that need clarification.

Thank you — once we receive the reviewed file we'll proceed with the extraction.

## CPT codes

**What we need from you:**

should be added but aren't listed, include them in the spreadsheet or note them in your reply.
Feel free to add comments for any codes that need clarification.

Thank you — once we receive the reviewed file we'll proceed with the extraction.

## Medications

We've generated a list of medications that may be relevant to your study. Please review the
